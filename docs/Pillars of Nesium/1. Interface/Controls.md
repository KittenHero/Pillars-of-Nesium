# Keyboard
## Utility
- Pause Menu: ESC
- Character Menus: Tab
- Look mode: U
- UI left: A
- UI right: D
- UI up: W
- UI down: S
- Confirm: Enter / Space
- Next tab: E
- Previous tab: Q
- Select tab: 1, 2, 3, 4, 5, 6 etc.
## Character
- Interact: E
- Move Left: A
- Move Right: D
- Up: W
- Crouch: S
- Jump: Space
- Wind Move: Shift
- Melee: J
- Wind Melee: K
